第一步：在官網獲取權限
登入你的 Hugging Face 帳號。

前往 black-forest-labs/FLUX.1-schnell 頁面。

你會看到一個黃色或藍色的框框，要求你接受條款。填寫簡單資訊後點擊 "Agree and access repository"。

第二步：建立 Access Token
點擊你的頭像 -> Settings -> Access Tokens。

點擊 "New token"，名字隨便取（例如 my-ai-agent），類型選擇 Read。

複製這個 Token，等一下會用到。

第三步：在本地端登入
回到你的終端機（MINGW64 / PowerShell），在你的專案目錄下：

PowerShell

from __future__ import annotations
import torch
import torch_directml
from diffusers import FluxPipeline

device = torch_directml.device()
print(f"🚀 正在使用裝置: {device}")

# 1. 載入模型時不直接 .to(device)
pipe = FluxPipeline.from_pretrained(
    "black-forest-labs/FLUX.1-schnell", 
    torch_dtype=torch.float16
)

# 2. 關鍵：使用這行代替 pipe.to(device)
# 這會自動管理 VRAM，雖然會慢一點點，但保證不會崩潰
# 刪除這行
# pipe.enable_sequential_cpu_offload() #default cuda NV GPU

# 換成這行
pipe.enable_model_cpu_offload(device=device)
print("🎨 正在生成圖片 (使用分流加載技術)...")

prompt = "A high-tech cyberpunk workstation with AMD GPU, 8k resolution"
image = pipe(
    prompt,
    guidance_scale=0.0,
    num_inference_steps=4,
    max_sequence_length=256,
).images[0]

image.save("flux_test_9070xt.png")
print("✅ 成功生成圖片！請查看資料夾。")

from __future__ import annotations
import torch
import torch_directml
from diffusers import FluxPipeline

device = torch_directml.device()
print(f"🚀 正在使用裝置: {device}")

# 1. 載入模型
pipe = FluxPipeline.from_pretrained(
    "black-forest-labs/FLUX.1-schnell", 
    torch_dtype=torch.float16
)

# 2. 針對 AMD DirectML 的手動優化
# 我們只把最耗算力的 transformer 搬到 GPU
# text_encoder 和 vae 留在 CPU (稍微慢一點，但保證不崩潰)
pipe.transformer.to(device)
# pipe.vae.to(device) # 如果顯存夠再開這行，不夠就註解掉

print("🎨 正在生成圖片 (手動 VRAM 分流模式)...")

prompt = "A high-tech cyberpunk workstation with AMD GPU, 8k resolution"
image = pipe(
    prompt,
    guidance_scale=0.0,
    num_inference_steps=4,
    max_sequence_length=256,
).images[0]

image.save("flux_test_9070xt.png")
print("✅ 成功生成圖片！")
